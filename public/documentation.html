Movie API

Purpose

This API is a project as part of my Full Stack Web Developer course at Career Foundry

It is an API to query for details about films

Standard Data Objects

There are a number of data objects used throughout this API which are outlined here so they can be abreviated later

Name Tag Description Format
Movie Data - Basic $movieBasic Basic data about a movie for large and initial queries {"movieId":[MID], "Title":[string], "Year":[yyyy]}
Movie Data - Detail $movieDetail Full detailed data about a movie {"movieId":[MID], "Title":[string], "Year":[yyyy], $genre, $director, "description":[string]}
Genre Data $genre Basic data about a genre {"genreId":[GID], "genre":[string], "movieCount":[int]}
Director Data $director Basic data about a director {"directorId":[DID], "director":[string], "yob":[yyyy], "yod":[yyyy], "movieCount":[int]}
User Info $userInfo Basic user info {"Username":[string], "Name":[string]}
